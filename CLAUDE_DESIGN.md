# עיצוב הפרויקט – הקצביה של פיני

## סקירה כללית

הפרויקט כולל שני ממשקים עיקריים:
1. **פאנל צ'אט** – ממשק שיחה עם לקוחות (WhatsApp AI Chat)
2. **פאנל אדמין** – ממשק ניהול עסקי עם גרפים, נתונים ועריכת תוכן

המשתמש יכול לעבור בין שני הפאנלים בקלות באמצעות כפתור/תפריט.

## פאנל צ'אט (Chat Panel)

### תיאור
ממשק שיחה פשוט ונקי עם Claude AI. הלקוח שולח שאלות ומקבל תשובות אוטומטיות.

### תכונות עיקריות
1. **ממשק הודעות** – בועות הודעה (משתמש מימין, AI משמאל)
2. **שדה קלט** – תיבת טקסט + כפתור שליחה
3. **Typing indicator** – אנימציה בזמן המתנה לתשובה
4. **ניקוי היסטוריה** – כפתור לאיפוס השיחה

### עיצוב
- **רקע:** `bg-[#0F0E27]`
- **הודעת משתמש:** `bg-[#7C5CBF] text-white rounded-2xl p-4`
- **הודעת AI:** `bg-[#1A1835] text-white rounded-2xl p-4 border border-[#2D2B52]`
- **שדה קלט:** `bg-[#1A1835] border border-[#2D2B52] rounded-xl px-4 py-3 text-white`

### תהליך שיחה טיפוסי
```
לקוח: "מה המחירים שלכם?"
AI: "שלום! אנחנו מציעים מגוון בשרים..."

לקוח: "בנה לי תפריט ל-10 אנשים ב-1000 שקל"
AI: "בטח! הנה הצעת תפריט:
     - אנטריקוט... ₪500
     - עוף שלם... ₪300
     ..."

לקוח: "אישור!"
AI: "מעולה! משלוח או איסוף עצמי?"
...
```

ממשק ניהול מקיף עם לוח בקרה, ניהול מוצרים, מבצעים, ועוד.

### מבנה ניווט

  1. 🏠 לוח בקרה (Dashboard)
  2. 💬 דוגמאות צ'אט (Chat Examples)
  3. 📝 תשובות מותאמות (Custom Answers)
  4. 🔄 סנכרון מוצרים (Sync Products)
  5. 🏷️ ניהול מבצעים (Discount Manager)
  6. 📦 ניהול מוצרים (Product Manager)
  7. ⚙️ הגדרות (Settings)

## 1. לוח הבקרה (Dashboard)

### רכיבים

#### KPI Cards (שורה עליונה)
ארבעה כרטיסי סטטיסטיקה:

**1. סה״כ הכנסות מהצ׳אט**
- מציג: `₪2,910`
- אייקון: 💰
- סטטוס: "מכל הזמנות הצ׳אט"

**2. שיעור המרה**
- מציג: `46.2%`
- אייקון: 📈
- סטטוס: "מתוך 13 שיחות"

**3. הזמנות שנסגרו**
- מציג: `6`
- אייקון: ✅
- סטטוס: "הזמנות מוצלחות"

**4. לקוחות חדשים**
- מציג: `1`
- אייקון: 👥
- סטטוס: "מהחודש האחרון"

#### גרפים (שורה שנייה)

**1. הכנסות בחודש האחרון (Area Chart)**
- **ציר X:** תאריכים (כל 3 ימים)
- **ציר Y:** סכום הכנסות (₪)
- **סגנון:** גרדיאנט סגול-טורקיז (`#7C5CBF` → `#00D4D4`)
- **נקודות:** נקודות בולטות בקו הגרף כל 3 ימים
- **רכיב:** Recharts `<AreaChart>`

**2. פילוח מכירות לפי קטגוריה (Pie Chart)**
- **קטגוריות:** בקר טרי, עוף, כבש, אחר...
- **ערכים:** סכום מכירות (₪) לפי קטגוריה
- **צבעים:** פלטת CHART_COLORS: `['#7C5CBF', '#9B7EE8', '#00D4D4', '#22C55E', '#F59E0B', '#EF4444']`
- **תצוגה:** עם Labels של שם + אחוז
- **רכיב:** Recharts `<PieChart>`

#### טבלת לקוחות (תחתית)

טבלה מלאה של כל הלקוחות:

| שם | מספר טלפון | סטטוס |
|----|-----------|-------|
| יוסי כהן | 972501234568 | 🟢 הזמנה נסגרה |
| דוד אברהם | 972501234569 | 🔴 אין מענה |
| מרים לוי | 972501234570 | 🟡 מחכה לתשובה |

**סטטוסים:**
- 🟢 **הזמנה נסגרה** – הזמנה הושלמה בהצלחה (`bg-[#22C55E]/10 text-[#22C55E]`)
- 🟡 **מחכה לתשובה** – שיחה פעילה, ממתין לתגובה (`bg-[#F59E0B]/10 text-[#F59E0B]`)
- 🔴 **אין מענה** – שיחה נסגרה בגלל חוסר מענה (`bg-[#EF4444]/10 text-[#EF4444]`)

**עיצוב:**
- רקע: `bg-[#1A1835]`
- שורה: `hover:bg-[#252246] transition-all`
- גבול: `border-[#2D2B52]`

## 4. סנכרון מוצרים (Sync Products)

### תיאור
כפתור להרצת Scrapers (Python scripts) שמעדכנים את:
1. `meatown_menu.json` – קטלוג מוצרים
2. `meatown_discounts.json` – מבצעים מהאתר

### UI
- **כפתור ראשי:** "🔄 סנכרון מוצרים מהאתר"
- **Spinner:** בזמן ריצה
- **לוג:** הצגת תוצאות הריצה (הצלחה/כשלון)
- **סטטוס אחרון:** "עדכון אחרון: 23/04/2026 14:30"

## 5. ניהול מבצעים (Discount Manager)

### תיאור
ניהול המבצעים המותאמים (נוסף למבצעים מה-Scraper).

### מבנה מבצע
```json
{
  "id": "uuid",
  "name": "מבצע שבועי – אנטריקוט",
  "description": "קילו אנטריקוט ב-₪200 במקום ₪270",
  "discount": "26%",
  "active": true
}
```

### פעולות
- **הוספה:** Modal עם שדות Name, Description, Discount%
- **עריכה:** עדכון מבצע קיים
- **מחיקה:** מחיקת מבצע מותאם
- **Toggle Active/Inactive:** כפתור on/off

### תצוגה
- **טבלה:** רשימת מבצעים עם עמודות (שם, תיאור, הנחה, סטטוס)
- **Badge:** `פעיל` / `לא פעיל`

## מעבר בין הפאנלים

### כפתור/תפריט
בחלק העליון של הממשק (Header):
- **כפתור Toggle:** "💬 צ׳אט" ↔ "⚙️ אדמין"
- **אנימציה:** Slide transition בין הפאנלים
- **State:** נשמר ב-`useState` ב-`App.jsx`

### דוגמת קוד
```jsx
const [view, setView] = useState('chat') // 'chat' | 'admin'

return (
  <div className="bg-[#0F0E27] min-h-screen text-white" dir="rtl">
    <Header view={view} onToggle={() => setView(v => v === 'chat' ? 'admin' : 'chat')} />
    {view === 'chat' ? <ChatPanel /> : <AdminLayout />}
  </div>
)
```

## הערות חשובות

1. **כל הממשק בעברית** – אין טקסט באנגלית ב-UI
2. **RTL בכל מקום** – `dir="rtl"` על כל הרכיבים
3. **עיצוב עקבי** – שימוש בפלטת הצבעים הקבועה
4. **נגישות** – ניגודיות טובה, גופנים קריאים
5. **ביצועים** – טעינת נתונים async, Spinner בזמן המתנה
6. **שגיאות** – Toast messages לכל פעולה (הצלחה/כשלון)
